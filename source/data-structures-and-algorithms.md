## 算法时间复杂度

公式：`T(n) = O(f(n))`

- n: 输入数据规模；
- f(n): 代码执行次数总和；
- T(n): 代码执行时间；
- O(f(n)): T(n) 与 f(n) 成正比；

时间复杂度分析方法：

- f(n) 中的低阶，常量、系数及低阶需要忽略；
- f(n) 中遇到相加时，选择最大的；
- f(n) 中遇到嵌套，需要相乘；

时间复杂度量级排序：

- O(1): 常量级，复杂度不随 n 增长；
- O(logn): 对数级
- O(n): 线性级
- O(nlogn): 线性对数级
- O(n2): 平方级
- O(n3): 立方级
- O(nk): K 次方级
- O(2n): 指数级
- O(n!): 阶乘级

空间复杂度量级排序：

- O(1)
- O(n)
- O(n2)
